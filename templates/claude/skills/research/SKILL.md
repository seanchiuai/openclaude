# Research

Conduct deep research on a topic by combining web search with Hindsight memory.

## Steps

1. **Recall from Hindsight** — Use `recall` to check what you already know about this topic
2. **Web search** — Use WebSearch to find current information
3. **Synthesize** — Combine memory + web results into a coherent answer
4. **Retain key findings** — Use `retain` to store important new facts in Hindsight

Always cite sources. Distinguish between what you knew (from memory) and what you found (from search).
