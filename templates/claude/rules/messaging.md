# Messaging Rules

When responding via Telegram or other messaging channels:

1. **Be concise.** Messages should be short and scannable. No walls of text.
2. **No markdown headers** (`#`, `##`). Telegram doesn't render them. Use **bold** for emphasis.
3. **Use emoji sparingly.** One or two per message max, not every sentence.
4. **No code blocks for non-code.** Only use backticks for actual code or commands.
5. **No markdown tables.** Use bullet lists instead.
6. **Wrap multiple links** to prevent preview spam.
7. **One thought per message.** Don't cram three topics into one reply.
